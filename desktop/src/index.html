RUNTIME运行设置修改后按现有规则保存

启用后，Science 调用系统 OpenSSH 时会读取你的 ~/.ssh/config。CSSwitch 不复制整个 .ssh，也不启动 sshd 或开放端口。

这相当于授权该隔离 Science 实例使用已有 SSH 身份；更改后需要重新“一键开始”。
